# Qt工业软件开发生态手册

> 一本开源、持续更新的工业软件实战指南，基于作者十余年Qt工业软件开发、开源项目维护与生态建设经验汇聚而成。

## 关于文档

本文档旨在系统梳理工业软件开发中的架构设计、界面框架、数据可视化、性能优化与工程实践，尤其聚焦Qt技术栈在工业场景下的应用。内容源于本人在以下开源项目中的长期实践与思考：

- SARibbon：这是我开源的一个`Qt Ribbon`界面框架，已应用于多个国产工业软件
- QWT：这是我在原Qwt6基础上扩展的一个项目，用于支撑快速科学工程绘图
- Data Workbench：这是我的一个开源数据处理与分析软件

本文档不追求面面俱到的教程，而是以问题驱动，分享从需求分析、架构设计到代码实现的经验。
